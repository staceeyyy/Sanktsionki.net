## Идея компании
После ухода многих международных брендов кроссовок (Nike, Adidas, New Balance, Jordan, Yeezy, ASICS и других) из России, покупка оригинальных вещей превратилась в квест с неоправданными переплатами и рисками подделок.

**Sanktsionki.net**  — это аналитическая платформа для поиска наиболее выгодных предложений на рынке оригинальных кроссовок. Система автоматически собирает данные с международных маркетплейсов и resale-площадок, анализирует стоимость моделей с учетом курса валют, комиссии байеров, доставки и логистики, после чего предлагает пользователю оптимальный вариант покупки.

## Механизм работы нашего сайта
*   Анализ данных российских платформ для покупки брендовыъх кроссовок (Avito, SneakerHead, Lamoda, CDEK.Shopping)
*   Отслеживание цен в реальном времени на российском офлайн-рынке.
*   Учет данных о комиссиях популярных сервисов доставки.

## Как мы будем собирать данные?
Данные собираются автоматически с помощью веб скрапинга. Для каждого источника — Avito, Lamoda, CDEK Shopping и др. создан отдельный scraper, который открывает страницы товаров, считывает информацию и извлекает данные о кроссовках: бренд, модель, цену, размер и ссылку на товар. Для работы с динамическими сайтами использовались библиотеки Selenium и BeautifulSoup. После сбора данные сохраняются в сырые датасеты, затем проходят этап очистки и нормализации: удаляются дубликаты, приводятся к единому формату названия моделей и рассчитывается итоговая стоимость с учетом курса валют, комиссии байеров и доставки.Эвристически анализируем чаты байеров в Telegram, извлекаем оттуда информацию о популярных моделях, спросе и рыночных ценах. После обработки все данные объединяются в аналитические таблицы.

## Целевая аудитория
B2C:
*   Люди, которые хотят купить оригинальный товар без 200% наценки перекупа.
*   Пользователи, которым важна оригинальность модели, но не хочется вручную сравнивать десятки площадок.
*   Те, кто не хочет нарваться на подделку на маркетплейсах.
  
B2B: 
*   Небольшие байеры
*   Реселлеры
*   Telegram-магазины и селлеры, которым нужен быстрый ориентир по рынку.
